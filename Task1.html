Key Definitions and Fundamentals: HTML and CSS

HTML (Hypertext Markup Language)

HTML serves as the foundation for crafting web pages. It structures and outlines webpage content using various elements and tags.

  • Element: An elemental unit of an HTML document, comprised of an opening tag, content, and possibly a closing tag.
  • Tag: A markup entity used to define content structure or format within HTML.
  • Attribute: Supplementary details or properties added to HTML elements to alter their behavior or appearance.
  • DOCTYPE: An introductory declaration in an HTML document, specifying the HTML version and activating standards mode for rendering.
  • Semantic HTML: HTML components that convey significance and organization to both browsers and developers, thus enhancing accessibility and SEO.

CSS (Cascading Style Sheets)

CSS acts as a styling language, dictating the presentation and layout of HTML documents. It dictates how elements are displayed across various mediums such as screen, print, or others.

  • Selectors: Patterns employed to target elements for CSS rule application.
  • Properties: Characteristics or styles applicable to selected elements, like color, font size, or margin.
  • Values: Specific settings or data assigned to CSS properties, determining their impact on element appearance.
  • External CSS: A standalone CSS file linked to an HTML document, facilitating centralized styling and reusability across multiple pages.
  • Inline CSS: CSS styles directly embedded within HTML elements through the style attribute.
  • CSS Box Model: The foundational layout structure in CSS, comprising content, padding, border, and margin surrounding elements.
